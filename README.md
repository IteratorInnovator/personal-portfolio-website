# Personal Portfolio Website

Modern, responsive portfolio for Harry Ng that showcases work experience, projects, skills, and certifications. The site pairs thoughtful storytelling with production-ready details like reproducible data structures, animated UI flourishes, and a Resend-powered contact pipeline.

## Features

- Landing hero with animated background beams, dynamic typewriter titles, and CTA buttons tied into the router.
- Data-driven sections for highlights, qualifications, skills, certifications (Embla carousel), and the new Projects hub with filtering + case study layout.
- GitHub activity analytics board with theme-sync, tooltip localization, and year switching.
- Contact page featuring validation, rich inputs, and a transactional email workflow (confirmation + notification) via Resend.
- Consistent dark/light theming handled through a custom toggle that syncs across components.

## Tech Stack

- **Frameworks & Libraries:** React 19, Vite, React Router, Motion, Embla Carousel, React GitHub Calendar.
- **Styling:** Tailwind CSS v4, custom CSS utilities, Lucide + React Icons.
- **Backend / APIs:** Vercel serverless functions, Resend email API.
- **Tooling:** ESLint, Tailwind CLI, npm scripts.
